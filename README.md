Interactive Power BI dashboard covering every Formula 1 season since 1950.
Based on a clean and regularly updated dataset maintained as part of the RaceOptiData project.

Live dashboard:
https://www.raceoptidata.com/f1_dashboard.html

Project website:
https://www.raceoptidata.com

Features
Season overview (drivers & constructors standings)
Race analytics: lap gaps, pace, progression, pit stops
Qualifying (Q1 / Q2 / Q3) analysis
Driver profiles & teammate comparison
Team performance analytics
Circuit map and multi-season navigation

Data source
Data is extracted and structured through a custom ingestion pipeline developed as part of RaceOptiData, providing an updated alternative to the historical Ergast dataset.
